Privacy Policy

Naruto Official

Effective Date: July 2, 2026

This Privacy Policy explains how Naruto Official ("the Application") collects, uses, stores, and protects information while providing services through Discord.

1. Information We Collect

To provide its functionality, the Application may collect and process:

  • User IDs
  • Guild (Server) IDs
  • Channel IDs
  • Role IDs
  • Configuration data required for server settings.
  • Moderation logs when enabled by server administrators.
  • Interaction metadata, including command usage and timestamps, to improve stability and diagnose issues.

2. Message Content

When authorized through Discord Privileged Intents, the Application may process message content exclusively to provide moderation, logging, automoderation, filtering, support, or other features explicitly configured by the server administrator.

Message content is processed only as necessary to perform these features and is not stored permanently unless a specific feature explicitly requires it.

3. How We Use Your Information

  • Provide bot functionality.
  • Maintain server configurations.
  • Provide moderation features.
  • Generate server logs when enabled.
  • Improve performance and reliability.
  • Prevent abuse and misuse of the Application.

4. Data Security

We implement reasonable administrative and technical measures to protect stored data against unauthorized access, disclosure, alteration, or destruction.

We do not sell, rent, or share personal information with third parties.

5. Data Retention

Data is retained only for as long as necessary to provide the services offered by the Application.

Removing the Application from a Discord server will result in the deletion of that server's stored configuration within a reasonable period, except where temporary retention is required for operational or legal purposes.

6. User Rights

Users and server owners may request deletion of stored information by contacting the development team.

7. Third-Party Services

The Application communicates only with the official Discord API and does not use advertising, tracking services, or third-party analytics platforms.

8. Changes to this Policy

This Privacy Policy may be updated periodically. Continued use of the Application constitutes acceptance of any future revisions.

9. Contact

If you have questions regarding this Privacy Policy or wish to request deletion of your data, you may contact the development team through our official support server:
